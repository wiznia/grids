GRIDS!

Grids are a new way of organizing a layout via CSS on a page

It allows us to define a structure in which you can control what goes into each row or column and in what order.

Standard grid

Grid lines

Grid lines are the lines that make the grid. They can be horizontals or verticals and we can call them by name or number.

Grid lines (highlighted)

Grid track

Grid Track is the space between two grid lines, horizontal or vertical.

Grid cell

Grid Cell is the space between 4 grid lines. Is the smallest unit in which we can place content.

Grid area

Grid Area is any area inside the grid that's contained between 4 grid lines. It can have multiple grid cells inside.
